## spark常用操作算子



#### 聚合

##### combineByKey

```scala
var rdd_x1_all = rdd_x1.combineByKey(
      List(_), //转换为list类型
      (x:List[(Int, Int, Int)], y:(Int, Int, Int)) => y :: x,   //将新元素y添加到x列表中
      (x:List[(Int, Int, Int)], y:List[(Int, Int, Int)]) => x ::: y //将key相同的连接起来，其实这里key相同的只存在一个分区里面。
    )
```

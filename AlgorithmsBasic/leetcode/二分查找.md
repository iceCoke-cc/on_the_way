##### 二分查找

- 明确变量的含义
- 循环不变量。即target永远在[l...r]中，我们需要维护的就是[l...r]这个数组（通过控制边界来达到目的）。

```c++
//二分查找
int binarySearch(T arr[], int n, T target){
  int l = 0; r = n - 1; //在 [l ... r]的范围里寻找target
  while( l <= r ){ //区间 [l ... r]依然有效
    int mid = (l+r)/2;
    if( arr[mid] == target )
      return mid;
    if(target > arr[mid])//排除mid, target在右侧，更新左边界
      l = mid + 1;
    else
      r = mid - 1;
  }
  return -1;
}
```
